# Medicine Tracker Backend

A TypeScript-based backend service for the Medicine Tracker application.

## Project Structure

```
backend/
├── src/
│   └── module/
│       └── user/
│           ├── userServices.ts
│           └── userDTO.ts
├── node_modules/
├── .env
├── .gitignore
├── package.json
├── tsconfig.json
└── README.md
```

## Prerequisites

- Node.js ≥ 18
- npm or yarn

## Setup

1. Clone the repository:

```bash
git clone git@github.com:AbuJubair7/medicine-tracker.git
cd backend
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file and add your environment variables.

4. Build and run:

```bash
npm run build
npm run start
```

## Development

```bash
npm run dev
```

## Testing

```bash
npm test
```
